# Nous v0.4.0 · 智性相伴，一路同行

> _从一个 AI 陪你想，到一整张网 · 六位智者 · 陪你把每个想法落到执行。_

自 **v0.3.0** 以来，Nous 跨越了六个大步，并在最后一轮里把所有体验细节又磨了一遍。
这是迄今为止覆盖面最广、也最能让你感受到"这个产品真的懂你"的一版。

---

## 核心：五项能力重塑

### 01 · 智性相伴 · 六位顶级智者

与其选择一种"AI 语气"，不如选一位**同行者**。

| 印章 | 人物 | 风格 |
|:-:|:--|:--|
| 羽 | **诸葛亮** | 谋士 · 文言温厚 · 可深思 |
| R | **Rick** | C-137 · *打嗝* · 虚无主义 · Wubba lubba dub dub |
| 楚 | **楚轩** | 《无限恐怖》· 代价必须合理 |
| Σ | **苏格拉底** | 反诘 · 我一无所知 |
| 鲲 | **庄子** | 寓言 · 相对 · 逍遥 |
| H | **Holmes** | Observation + Deduction · 排除不可能 |

每位智者都有**自己的朱砂印章、语音层、提问方式、方案拟定视角**——
从工作台问候，到 4 阶段 Socratic 对话，到方案生成，到主动提问，
他会贯穿全流程，用自己的思维方式陪你走完这一轮。

> 新手引导弹窗已加入这一步 · 每个印章会逐格缓出。

### 02 · Proactive · AI 主动问你

不再等你来问。Nous 现在会在合适的时机**主动敲一下门**：

- **Zombie Idea** — "你说过想做这件事 · 搁置一周多了 · 还想继续吗？"
- **Stalled Plan** — "这个方案上次推进是 10 天前 · 要不要调整目标？"
- **Hoarding Pattern** — "这周记了 14 个新想法 · 只深入了 2 个。要不要先收拢？"
- **Orphan Goal** — "你提过想……但它还没有 plan · 要不要聊聊？"
- **Dormant Blindspot** — "上次我们提过 X · 你后来想清楚了吗？"
- **Seasonal Review** — 每周日 · 新月 · 节气转折，主动邀你复盘

每条问卡由你当前选定的**智者重写**成他的口吻。
支持一周内跳过同 key · 防止打扰疲劳。

---

### 03 · Chronicle · 你的思维时间线

一屏看到你**所有想过的、做过的、记过的事**。

- 想法落笔 / 笔记更新 / 对话轮次 / 复盘生成 / 记忆沉淀 / 任务完成
- 日 · 周 · 月 三维切换
- 按 `YYYY-MM-DD` 精准分组，空日自动跳过
- **每条 AI 回应显示是哪位智者说的**（"诸葛亮 回应" / "Holmes 回应" / …）
- 点任意一行直接跳去源头

`/chronicle` 现在是继 `/workspace` 之后，最有"长期感"的一个入口。

---

### 04 · Omni-Search · 一次搜五类

`⌘⇧K` 或 `⌘K` 输入 `?` 前缀进入深度搜索，**同时扫描**：

想法 · 笔记 · 消息 · 复盘 · 记忆

混合全文检索 + 向量相似，相关度打分排序 · 高亮命中片段。

---

### 05 · RelationRail · 想法不再孤岛

每个想法 / 笔记 / 复盘，右侧都有一条**关联栏**：

- 自动链出同主题的想法、你写过的相关笔记、AI 沉淀的记忆
- 任何方向都可双向跳转
- 从今往后，你在 Nous 里写的东西 · 不会只是一堆孤立的便签

## 体验细节升级

### 落一笔 · 从想到记一步到位

工作台右上"+ 落一笔"按钮**直连捕获模式**，输入即想法内容，不再经过搜索栏。
旁边 `⌘K` 副键仍保留命令面板入口，分段按钮，职能清晰。

### 一键聚焦 · 方案到行动的直线

方案页在"第一步"卡片后新增 **"聚焦 N 件"** 朱砂横条，
一键把本方案所有必做任务加入今日聚焦，省下 N 次点击。

### 对话起手不再发懵

新对话进 refine 不再只有空白文本框。现在会告诉你：

> _"我会用四个阶段问：意图 → 细节 → 边界 → 就绪。"_
> _"先说你现在脑子里最清楚的部分 — 也可以挑一句开头。"_

附 **3 个起手句按钮**，点击即填入输入框。
输入框 placeholder 也根据当前 phase 实时变化——从"先说说你最想先确定的那一步"到"准备生成方案了 · 也可以再补一句再生成"。

### 每个想法 · 记得它自己的那位智者

- 新对话默认 **Nous 本体**
- 有历史对话的想法 · 自动用上次选的智者
- 切智者不再被"别的想法曾经的选择"打扰 —— **per-idea 记忆**

### Plan 生成也带智者视角

生成 / 重生方案时会带上你当前的智者 id，
不同智者拟出的目标语气、里程碑节奏、第一步文案都不同。
方案页表头会显示"由 诸葛亮 拟定"一行。

### 工作台问候 · 时段 × 智者 × 你今日状态

`GreetingCard` 融合：你选的智者 + 当前时段 + 今日完成 / 本周想法 / 连续天数
生成一句不重样的开场问候。无 `Date.now()` 抖动，SSR 稳定。

### ⌘K · 无命中也能用

输入一句话、没任何命中？直接 Enter，**整句落成新想法**，进 refine 继续聊。
之前那句"按 Enter 落成新想法"的提示，现在终于说到做到。

### 智者选择器 · 优雅入场

Refine 页的智者下拉改为 **0.18s CSS `dropIn`**，从右上角自然落下，
按钮文字从"语气"升级为"当前智者"，语汇与 Onboarding 一致。

## 兼容性与升级

从 v0.3.0 升级，建议顺序：

```bash
git pull
pnpm install
pnpm prisma migrate deploy   # stage1-6 带来的 schema 扩展
pnpm build
pnpm start
```

本次**无破坏性 API 变更**，但：

- 历史 `assistant` 消息无 `personaId` 字段 · 会在 Chronicle / 方案页显示为"AI 回应" fallback，不影响功能
- 如你之前给 demo 账号 seed 过 notion 示例想法，请参考 `prisma/seed.ts` 新版清理重复数据
